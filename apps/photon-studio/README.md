# Photon Studio

Archived browser image-editor prototype.

## Files

- [`index.html`](./index.html) is the archived editor entry page.
- [`studio.css`](./studio.css) contains the editor styling.
- [`src/editorApp.js`](./src/editorApp.js) contains the editor UI.
- [`src/editorState.js`](./src/editorState.js) contains pure editor-state helpers.
