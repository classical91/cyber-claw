# Report Studio

Archived transcript-to-report planning prototype.

## Files

- [`index.html`](./index.html) is the archived report studio entry page.
- [`strategy.css`](./strategy.css) contains the app styling.
- [`src/reportStudioApp.js`](./src/reportStudioApp.js) renders the studio UI.
- [`src/reportEngine.js`](./src/reportEngine.js) contains report-planning logic and scoring helpers.
